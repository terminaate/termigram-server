# Branch architecture

```
core/* - branch с какими-то глобальными изменениями, такие как изменение системы авторизации, стека приложения и прочие
feature/* - branch с какими-то второстепенными изменениями, тоесть например добавление новой функциональности
fix/* - branch с какими-то правками кода, тоесть допустим устранением незначительных багов
hotfix/* - branch с какими-то фиксами серьёзных багов
```
